# Counter Dashboard

A modern React Counter Dashboard built with React, Tailwind CSS, and Local Storage. The application allows users to manage a counter, track recent activities, and switch between Light and Dark themes with persistent state.

![Light Mode](./screenshots/light-mode.png)

## Features

* Increment Counter (+1)
* Decrement Counter (-1)
* Increment Counter by 5 (+5)
* Reset Counter
* Activity History Tracking
* Display Latest 5 Activities
* Clear Activity History
* Light & Dark Theme Toggle
* Persistent Theme Preference
* Persistent Counter State
* Persistent History State
* Responsive and Modern UI
* Built with Tailwind CSS

## Tech Stack

* React
* Tailwind CSS
* JavaScript (ES6+)
* Local Storage API

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd counter-dashboard
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
