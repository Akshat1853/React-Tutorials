# FocusFlow - Smart Todo Manager

A modern Todo application built with React and Tailwind CSS to help users organize daily tasks efficiently.

## Features

### Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed or pending

### Search & Filtering

* Search tasks instantly
* Filter tasks by:

  * All
  * Pending
  * Completed

### Statistics Dashboard

* Total tasks count
* Completed tasks count
* Pending tasks count

### Local Storage Persistence

* Tasks are automatically saved in the browser
* Data persists even after refreshing the page

## Tech Stack

* React
* JavaScript (ES6+)
* Tailwind CSS
* Local Storage API
